Data Mining Project

Title: Creating Awareness of Road Safety in Queensland


Website

This is my data mining project from when I was studying at University of Queensland. Queensland Government has released a dataset which contains information about car crashes since 2001. The dataset showed that around 12,000 road accidents happen yearly in Queensland, and since 2001 over 4000 of them have been fatal. Therefore, the project sought to discover interesting patterns in this dataset and to raise public awareness of the factors that increase the severity of accidents.
As said the data was provided from the Queensland government and had many different attributes such as female involvement, the location of the crash, the number of people involved, etc. For initial testing of the data, the data was processed for using it in WEKA (Waikato Environment for Knowledge Analysis). WEKA helped us experiment with different data mining classifiers/association rules and testing different combination of attributes to discover which model performed best.
The model that performed the best in our case was Na√Øve Bayes and we used this model in the website.
